# AXIOM Architecture

## Verification Pipeline

```
Input Code → Assertion Checker → Logic Analyzer → Invariant Detector
    → Proof Generator → Boundary Analyzer → Concurrency Verifier
    → Specification Validator → Termination Prover → Abstract Interpreter
    → Verification Report → Proof Summary
```

## Token Model

- Per-verification: ~191K tokens (all 10 agents)
- Daily: ~82M tokens (500 verifications)
- Monthly: ~2.47B tokens
